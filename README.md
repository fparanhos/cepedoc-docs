## Como contribuir

Pull requests são bem-vindos. Para correções no manual:

1. Faça fork do repo
2. Edite o arquivo `.md` correspondente
3. Abra um PR descrevendo a mudança

Para sugestões sem PR, abra uma [Issue](https://github.com/fparanhos/cepedoc-docs/issues).

## Renderização dos diagramas

Os arquivos `.md` deste repositório usam **Mermaid** para diagramas de fluxo. O GitHub renderiza Mermaid automaticamente na visualização web — basta abrir o arquivo aqui.

Para visualizar localmente, recomendamos:
- **VS Code** com a extensão "Markdown Preview Mermaid Support"
- **Obsidian** (suporte Mermaid nativo)
